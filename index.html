☁️ Sincronizzazione Google Sheets (Opzionale)

Lascia vuoto per usare solo il salvataggio locale
📋 Come configurare Google Sheets:
1. Crea un nuovo Google Sheets
2. Vai su "Condividi" → "Chiunque con il link può visualizzare"
3. Copia il link e incollalo qui sopra
4. Testa la connessione

⚠️ Limitazione CORS:
I browser bloccano l'accesso a Google Sheets da file locali per sicurezza.
🛠️ Soluzioni CORS:
1. Hosting Web (Consigliato):
• Carica su GitHub Pages, Netlify, Vercel (gratis)
• CORS problema risolto automaticamente

2. Server Locale:
• Terminal: python -m http.server 8000
• Apri: http://localhost:8000/travel-list.html

3. Sync Manuale (Sempre Funziona):
• Usa Export/Import per multi-device
• Zero problemi tecnici
• Massima affidabilità
